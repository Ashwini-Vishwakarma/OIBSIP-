# OIBSIP – Data Analytics Internship

This repository contains all the projects completed during the Oasis Infobyte Data Analytics Internship.

## Projects

### Task 1 – Retail Sales Exploratory Data Analysis
- Performed exploratory data analysis on a retail sales dataset.
- Analyzed sales trends, customer demographics, and product performance.
- Created various visualizations using Matplotlib and Seaborn.

### Task 2 – Customer Segmentation using K-Means
- Performed customer segmentation using RFM (Recency, Frequency, Monetary) analysis.
- Applied StandardScaler and K-Means clustering.
- Used the Elbow Method to determine the optimal number of clusters.
- Generated customer profiles and marketing recommendations.

### Task 3 – Data Cleaning
- Performed professional data cleaning on the Titanic dataset.
- Handled missing values, duplicate records, outliers, and inconsistent formatting.
- Produced a clean, analysis-ready dataset.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
