# Customer Segmentation Analysis using K-Means

## Project Overview

This project applies K-Means Clustering to segment customers based on their purchasing behaviour using RFM (Recency, Frequency, Monetary) analysis.

## Dataset

Online Retail Dataset (UCI / Kaggle)

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow

- Data Cleaning
- Feature Engineering
- RFM Analysis
- StandardScaler
- Elbow Method
- K-Means Clustering
- Cluster Visualization
- Customer Profiling
- Marketing Recommendations

## Business Insights

- Loyal customers identified
- At-risk customers identified
- High-value customers identified
- Marketing strategy suggested for each segment
