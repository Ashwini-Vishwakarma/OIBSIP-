Transform a messy dataset into a clean, analysis-ready dataset.

Titanic Dataset

## Cleaning Steps

- Missing value handling

pandas==3.0.3

## Dataset

This project uses the **Titanic Dataset**, which contains information about passengers aboard the RMS Titanic. The dataset includes features such as passenger class, age, sex, fare, embarkation port, and survival status.

**Source:** Kaggle – Titanic Dataset  
https://www.kaggle.com/datasets/yasserh/titanic-dataset

### Dataset Features
- PassengerId
- Survived
- Pclass
- Name
- Sex
- Age
- SibSp
- Parch
- Ticket
- Fare
- Cabin
- Embarked

> **Note:** The dataset is not included in this repository due to licensing and file size considerations. Please download it from the Kaggle link above and place the `Titanic-Dataset.csv` file in the project directory before running the notebook.
