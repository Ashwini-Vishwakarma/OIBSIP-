# Retail Sales Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a Retail Sales dataset using Python. The objective is to understand customer demographics, sales trends, product category performance, and generate actionable business insights through data visualization.

## 📂 Dataset

The dataset contains **1000 retail transactions** with the following features:

- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount

## 🛠️ Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Analysis Performed

- Initial Dataset Inspection
- Descriptive Statistics
- Monthly Sales Trend
- Quarterly Sales Trend
- Customer Demographics Analysis
- Product Category Analysis
- Correlation Heatmap
- Additional Visualization
- Business Recommendations

## 📈 Visualizations

- Line Charts
- Bar Charts
- Histogram
- Count Plot
- Heatmap

## 💡 Key Insights

- Customer demographics were analyzed using age groups and gender.
- Monthly and quarterly sales trends were identified.
- Product category revenue and quantity sold were compared.
- Correlation between numerical variables was visualized using a heatmap.

## ✅ Business Recommendations

1. Focus marketing efforts on the highest revenue-generating product category.
2. Target the most active customer age groups with personalized promotions.
3. Use monthly and quarterly sales trends to improve inventory planning.

## 🚀 How to Run

1. Clone the repository.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Open the notebook:

```bash
jupyter notebook Retail_Sales_EDA.ipynb
```
