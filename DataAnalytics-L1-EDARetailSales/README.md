# Retail Sales Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a Retail Sales dataset using Python. The objective is to analyze customer behavior, sales trends, and product performance through statistical analysis and data visualization.

## Dataset

The dataset contains 1000 retail transactions with the following columns:

- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed

- Data Inspection
- Descriptive Statistics
- Monthly Sales Trend
- Quarterly Sales Trend
- Customer Demographics Analysis
- Product Category Analysis
- Correlation Heatmap
- Business Recommendations

## Key Insights

- Electronics generated the highest revenue.
- Most customers belong to the 26–35 age group.
- Sales varied across different months.
- Quantity and Total Amount showed a strong positive correlation.

## Business Recommendations

1. Increase inventory for high-performing product categories.
2. Target the most active customer age groups with promotions.
3. Improve sales of lower-performing categories using discounts.
